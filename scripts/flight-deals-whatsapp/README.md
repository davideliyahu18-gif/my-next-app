# בוט דילי טיסות — מעקבים קבועים

1. **תאילנד** — TLV→BKK, `10/02/2027–10/03/2027`, יציאה **15:10→07:35**, חזרה בלילה, **אמירטס + מזוודה**
2. **בודפשט** — TLV→BUD, `11/11/2026–15/11/2026`, מחירים ב־₪

## מקור מחירים

מומלץ: **Travelpayouts / Aviasales** (חינם, מכסה גבוהה)  
1. הרשמה ב־https://www.travelpayouts.com/  
2. חיבור לתוכנית Aviasales  
3. העתקת API token ל־`.env.local` כ־`TRAVELPAYOUTS_TOKEN=...`

SerpAPI נשאר אופציונלי לאימות עמוק (אמירטס+מזוודה), אבל נגמר מהר בחינם (250/חודש).

## הפעלה חד-פעמית

1. ודא ב-`.env.local`:
   - `SERPAPI_API_KEY=...`
   - `WHATSAPP_GROUP_CHAT_ID=120363428169341488@g.us`
   - `WHATSAPP_PHONE=` המספר שמקושר לבוט
2. הפעלה:
   ```bash
   npm run flight-deals:start
   ```
3. אם צריך חיבור מחדש (פעם אחת):
   ```bash
   npm run flight-deals:relink
   npm run flight-deals:start
   ```
   ואז סרוק QR מהמחשב עם הטלפון.

**חשוב:** אל תריצו שני מופעים במקביל — יש נעילה (`bot.lock`).

## בוט אישי (פרטי)

כל אחד שולח הודעה *ישירות לבוט* (לא בקבוצה):

| הודעה | מה קורה |
|--------|----------|
| `התחל` | הצטרפות למעקב אישי |
| `עצור` | ביטול התראות |
| `מחפש` / `סטטוס` | חיפוש מחיר עכשיו בפרטי |
| `תקציב 3200` | תקרת מחיר בשקלים |
| `רק תאילנד` / `רק בודפשט` / `הכל` | בחירת יעדים להתראות |
| `יעדים` | מה פעיל אצלך עכשיו |
| `עזרה` | תפריט |

כשהמחיר יורד (תאילנד או בודפשט) — ההתראה מגיעה **רק אליו בפרטי**.
